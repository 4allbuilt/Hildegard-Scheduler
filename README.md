# Professor Scheduler

교수님을 위한 세션 기반 업무 관리 PWA

## 주요 기능

### ✨ 핵심 디자인
- **세션 기반 시간 관리**: 아침(06:00-08:30), 오전(09:30-13:00), 오후(14:00-18:00)
- **5개 업무 영역**: 강의 준비, 연구개발, 학술논문, 저서작성, 교육프로그램
- **시각적 진행도**: 실시간 완료율 표시
- **자동 수업 스케줄**: 목요일/금요일 오후 수업 자동 반영

### 📱 사용 방법
1. 홈화면에서 세션 선택
2. 할 일 추가 (업무 영역별 분류)
3. 타이머로 집중 작업
4. 완료 체크로 진행률 확인

### 🎯 특징
- **단순함**: 오늘 할 일만 보여줌
- **시각적**: 진행률 바, 완료 애니메이션
- **편리함**: 홈화면 앱으로 설치

## 개발 환경 설정

```bash
# 패키지 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 배포 (Vercel)

1. GitHub에 리포지토리 생성
2. Vercel 계정 연동
3. 자동 배포 완료
4. 모바일에서 홈화면 추가

## 기술 스택

- React 18
- Vite
- Tailwind CSS
- PWA 지원
